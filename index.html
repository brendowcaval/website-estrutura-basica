Ferramentas utilizadas

Nesta parte é onde se prepara para utilizar o HTML, o ambiente para executá-lo. Existem diversas ferramentas para executar o HTML, alguns são bastante conhecidos, como o Visual Studio Code.

Inspetor de Elementos

As vezes é necessário visualizar e entender as tags de determinada página, numa página web, você clicando em Configurações>Ferramentas>Ferramentas de Desenvolvedor, você pode visualizar as tags daquela página e até manipular conforme a sua vontade, isso é bastante utilizado entre os Desenvolvedores.

Estrutura básica do HTML

Uma estrutura HTML contém sempre uma tag html, e dentro desta tag, existe tags como a head, que é o cabeçalho, onde é necessário o carregamento dela para exibir a página, e a tag body, que fica visivel para o usuário.

Tags

Uma tag é composta por inicialização e fechamento de tag, algumas tags não precisam ser finalizadas. e dentro de uma tag sempre tem algum contéudo.

Atributos das tags

Algumas tags existem atributos para representar, para determinar alguma funcionalidade.

Textos

As tags de textos, que como o nome já diz, é para exibir textos na página, como as tags h para exibir titulos, p para exibir parágrafos, e entre outras.

Listas ordenadas e não ordenadas

São tags para determinar uma lista, podendo ela ser ordenada(seguindo uma ordem) e não ordenada(sem ordem).

Links

Neste tipo de tag, sua função é navegação de páginas, podendo uma página ser redirecionada para outra.
